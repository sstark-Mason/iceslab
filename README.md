# iceslab

## Usage

Download && install:

```bash
sudo mkdir /opt/iceslab && cd /opt/iceslab \
&& curl -O https://raw.githubusercontent.com/sstark-mason/iceslab/main/iceslab \
&& chmod +x iceslab && ./iceslab -i`
```

Install:

`./iceslab -i`

Update:

`./iceslab -u <b, s>`

Dump:

`./iceslab --dump`
