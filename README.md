# iceslab

## Usage

Download && install:

```bash
sudo mkdir /opt/iceslab && cd /opt/iceslab \
&& sudo curl -O https://raw.githubusercontent.com/sstark-mason/iceslab/main/iceslab \
&& sudo chmod +x iceslab && sudo ./iceslab -i
```

Install:

`sudo ./iceslab -i`

Update:

`sudo ./iceslab -u <b, s>`

Dump:

`./iceslab --dump`
